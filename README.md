
 Task 11: Phishing Attack Simulation & Detection

 Objective
To simulate a phishing attack using GoPhish and analyze how users interact with phishing emails.

 Tools Used
- GoPhish
- Kali Linux
- Web Browser

 Steps Performed
1. Created a fake phishing email template.
2. Set up a phishing landing page.
3. Launched a phishing campaign.
4. Tracked email opens, clicks, and credentials.
5. Identified phishing red flags.
6. Learned prevention techniques.

 Outcome
Improved understanding of phishing attacks and social engineering awareness.
